# Notepad Clone

This is simple clone of the default notepad in windows 10.

____

## Missing features
  - Print
  - status bar (open to contributions)

___
## external package 
  - tkfontchooser

    Can be installed by running

    ```bash
    pip install tkfontchooser
    ```

____
#### Run the program using the command
```bash
python main.py
```
